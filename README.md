## 🛠️ Installation & Setup

### 1. Clone the repo

```bash
git clone https://github.com/dhrumil0333/Korat-Dhrumil-Portfolio.git
cd portfolio
```

### 2. Install dependencies

``` bash
npm install
```

### 3. Start development server
``` bash
npm run dev
```

The site will run on http://localhost:5173
